Follow these steps to set up your Student Complaint Dashboard

Step 1: Start XAMPP

Open XAMPP Control Panel and start Apache and MySQL services.

Step 2: Access phpMyAdmin

Open your browser and navigate to:

http://localhost/phpmyadmin

Step 3: Create Database

1. Click on "New" in the left sidebar
2. Enter database name: student_complaints_db
3. Click "Create"

Step 4: Run SQL Script

1. Click on the newly created student_complaints_db
2. Go to the "SQL" tab
3. Open this file: backend/database/schema.sql
4. Copy all contents and paste into the SQL box
5. Click "Go"
